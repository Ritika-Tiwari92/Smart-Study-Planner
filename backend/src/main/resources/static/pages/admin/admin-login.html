🔑 Reset Password

Admin password reset requires direct database or system access for security reasons. Follow these steps to reset your admin password:

1

Open MySQL Workbench

Connect to your StudyPlanner database
2

Generate BCrypt Hash

Visit bcrypt-generator.com → enter new password → copy hash
3

Run SQL Update

UPDATE users SET password='<hash>' WHERE role='ADMIN';
4

Login with new password

Restart backend if needed, then sign in
