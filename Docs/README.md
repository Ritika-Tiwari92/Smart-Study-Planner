# 🚀 Smart Study Planner (Astra AI Powered)

## 📘 Project Overview
Smart Study Planner is an AI-powered web application designed for Computer Science students to manage their studies efficiently. It helps users plan their schedule, track progress, identify weak subjects, and improve productivity using intelligent suggestions.

---

## 🎯 Key Features

- 📚 Subject Management (DSA, DBMS, OS, CN, etc.)
- 🧠 Weak Subject Detection
- 📅 Smart Study Scheduler
- 🔁 Revision Planner (1, 3, 7 days system)
- 📝 Test & Quiz Scheduling
- 💻 Coding & DSA Tracker
- 📊 Performance Analytics (charts & progress)
- 🤖 Astra AI Assistant (personal study guide)
- 🎨 Modern UI with Midnight Teal Neon Theme
